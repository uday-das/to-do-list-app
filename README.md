# To Do List
<h2>About The Project </h2> 
<p>A web application that helps users manage their tasks by listing all the tasks that needs to be completed, organized in order of priority. In order to use this application, users must first register themselves. They can then add tasks to the list by clicking the "+" button or the "New Task" button. To add a task, enter the task title in the designated box and click on the "Submit" button. Once a task has been completed, the user can strike it off the list by checking the "Complete" checkbox. </p>

<h2>Tech Stack</h2> 
<p>Client Side: HTML, CSS, JavaScript </p>
<p>Server Side: Django</p>
<h2>How To Use This App</h2>
<ul>
<li> Clone this repository to your machine </li> 
<li> Install django if you haven't installed yet. </li> 
<li> Use the command pip install django to install it. </li> 
<li> Now navigate to the project directory and start the Django server by typing the python command followed by manage.py runserver command in the terminal. </li>
<li> This will start the development server on your local machine. </li> 
</ul>

<p> All Set! Your app should be now running on port:8000. Head over to your browser and enter localhost:8000 in the serach box and bang! You will see the application running in your browser.
